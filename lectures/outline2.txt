+ The Scariest Cautionary Tales: Disassembly
  ** I asked Claude CLI how to run it in a few ways
  ** It immediately started disassembling the binary
  ** It did not find the answer but it came close
  -- I then pointed out it could simply read the manuals
  -- If these agents get better even compiled code is at huge risk.
  
+ The Scariest Cautionary Tales: Open Source = Open Holes
  ** Popular package LiteLLM was hacked
  ** Node.js
  ** 40,000 users
  -- At this point you simply can't rely upon the security of
   open source packages.
  -- Not only are they heavily targeted but one can inject hidden
   code or functionality.
  -- Current AIs can simply reimplement them for you in a few hours, so
    you should to seal your security.
